# Catador de alfajores

Blog sobre alfajores argentinos, migrado de WordPress a Jekyll para GitHub Pages.

## Estructura

- `_posts/` - Posts del blog en formato Markdown
- `assets/images/` - Imágenes del blog (todas comprimidas a < 1MB)
- `_config.yml` - Configuración de Jekyll
- `index.md` - Página principal

## Desarrollo local

1. Instalar Ruby y Bundler
2. Instalar dependencias: `bundle install`
3. Ejecutar servidor local: `bundle exec jekyll serve`
4. Abrir en navegador: http://localhost:4000
